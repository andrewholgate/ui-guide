Introduction

This website contains a set of guidelines for WFP Communications Division, which are to be used throughout all WFP digital channels, such as websites, web applications, internal systems and other, including digital documents.

These guidelines are set for fellow content creators at WFP, to help them create consistent and beautiful content on the web in an easy way.
