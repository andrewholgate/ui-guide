Buttons

We use several buttons for you to choose from, make sure you apply only styles intended for a given element, to make sure it doesn't stand out too much or doesn't bear a different meaning.

Primary Button

Primary buttons are only used when there is a clear need for a call-to-action. An example of this is a Send button on an contact form, or a Subscribe button on a newsletter subscription form.

<button class="pure-button pure-button-primary">Button</button>
<button class="pure-button pure-button-primary pure-button-active">Button</button>
<button class="pure-button pure-button-primary pure-button-disabled">Button</button>
Secondary Button

Secondary buttons are used as a general UI block, for pretty much anything you need.

<button class="pure-button pure-button-secondary">Button</button>
<button class="pure-button pure-button-secondary pure-button-active">Button</button>
<button class="pure-button pure-button-secondary pure-button-disabled">Button</button>
